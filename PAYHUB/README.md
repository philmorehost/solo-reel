# Payhub PHP Version

This is a professional PHP-based conversion of the Payhub payment gateway platform.

## Features
- **Merchant Dashboard**: Overview, Transactions, Virtual Accounts, Payouts, Compliance, Settings, API Keys.
- **Admin Dashboard**: Platform management, Merchant directory, Settlement processing, Global configuration.
- **Authentication**: Secure login, registration, password reset.
- **Installer**: Multi-stage web installer for easy deployment on cPanel/Shared hosting.
- **Database**: MySQL with PDO for secure data handling.
- **UI/UX**: 100% fidelity to the original design using Tailwind CSS and Alpine.js.

## Installation
1. Upload the `php-version/` folder to your server.
2. Navigate to `yourdomain.com/install/` in your browser.
3. Follow the multi-stage installer to set up your database and admin account.
4. Delete the `install/` folder after successful installation.

## Requirements
- PHP 8.1+
- MySQL 5.7+
- PDO MySQL Extension
- OpenSSL Extension
- MBString Extension
